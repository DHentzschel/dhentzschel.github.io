SimpleLibs out now!

September 20, 2018


A new library project named SimpleLibs which was initiated to improve the programmer's workday is in the wings. It includes diverse sub libraries e.g. Network, MySql and Graphics. Give it a try and download the binaries or fork me on github to work on it yourself.

Both the source codes and the binaries are available on GitHub, give it a try and have a look at the README.md file positioned in the repository.
The library itself is written in C++ STL and WINAPI an may be cross platform compatible some time, I would appreciate your help.
Fork me on GitHub and use your chance.

Beginning to enhance some important standard classes such as std::string, std::list and std::vector, SimpleLibs gives a good possibility to work as easy as in other high-level programming languages.


It does not bother anymore to e.g. having to read a file and deserialize the input, as you can see, this task is as easy as never before.
These functions have been simplified comparable to C# or Java. Looks good, doesn't it?
